Task 2 - SQL Operations (Library Management System)

Objective
This project demonstrates basic SQL operations including:

- CREATE TABLE
- INSERT INTO
- UPDATE
- DELETE
- SELECT
- NULL Values
- PRIMARY KEY
- FOREIGN KEY

Database Tables

Categories
Stores different categories of books.

Books
Stores book information such as title, ISBN, publication year, and category.

Students
Stores student details.

IssuedBooks
Stores information about books issued to students.

SQL Operations Performed

 1. CREATE TABLE
Created the following tables:
- Categories
- Books
- Students
- IssuedBooks

 2. INSERT INTO
Inserted sample records into all tables.

 3. NULL Values
Used NULL values for:
- Book ISBN
- Student Email
- Due Date

4. UPDATE
Updated:
- Student email address
- Book publication year

5. DELETE
Deleted one issued book record.

6. SELECT
Displayed all records from the tables to verify the operations.

Concepts Used

- Relational Database Design
- Primary Keys
- Foreign Keys
- Data Manipulation Language (DML)
- Data Definition Language (DDL)

 Tools Used

- DB Fiddle
- SQL

 Output

Successfully performed Create, Insert, Update, Delete, and Select operations on a Library Management System database.
